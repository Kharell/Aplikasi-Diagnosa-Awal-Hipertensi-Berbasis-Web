Naive Bayes adalah algoritma machine learning sederhana yang digunakan untuk mengklasifikasikan data berdasarkan peluang (probabilitas).
Disebut naive (sederhana) karena algoritma ini menganggap setiap faktor input (seperti tekanan darah, pola makan, riwayat keluarga, dan keluhan) tidak saling memengaruhi satu sama lain.

Dalam konteks diagnosa hipertensi, Naive Bayes menghitung peluang pasien mengalami kategori:

- Tekanan darah rendah,

- Normal,

- Pre-hipertensi, atau

- Hipertensi,

berdasarkan data yang pasien masukkan ke sistem.
Hasil dengan nilai peluang terbesar akan menjadi diagnosa akhir.

3.Hasil dengan nilai peluang terbesar dipilih.

4. Hasil tersebut disimpan ke database dan ditampilkan di modal hasil diagnosa.

Berikut bagian kode yang menjalankan algoritma Naive Bayes:

function naiveBayesDiagnosa($sistol, $diastol, $pola_makan, $riwayat_keluarga, $keluhan) {
    // Daftar kategori tekanan darah
    $kelas = ['Rendah', 'Normal', 'Pre-Hipertensi', 'Hipertensi'];

    // Probabilitas awal (semua sama)
    $prob = array_fill_keys($kelas, 1.0);

    // Faktor 1ï¸âƒ£: Tekanan darah
    if ($sistol < 90 || $diastol < 60) {
        $prob['Rendah'] *= 0.8;
    } elseif ($sistol < 120 && $diastol < 80) {
        $prob['Normal'] *= 0.8;
    } elseif (($sistol >= 120 && $sistol <= 139) || ($diastol >= 80 && $diastol <= 89)) {
        $prob['Pre-Hipertensi'] *= 0.8;
    } else {
        $prob['Hipertensi'] *= 0.8;
    }

    // Faktor 2ï¸âƒ£: Pola makan
    if (strpos($pola_makan, 'asin') !== false || strpos($pola_makan, 'garam') !== false) {
        $prob['Hipertensi'] *= 1.3;
    }
    if (strpos($pola_makan, 'gorengan') !== false) {
        $prob['Pre-Hipertensi'] *= 1.2;
    }
    if (strpos($pola_makan, 'sayur') !== false || strpos($pola_makan, 'buah') !== false) {
        $prob['Normal'] *= 1.2;
    }

    // Faktor 3ï¸âƒ£: Riwayat keluarga
    if ($riwayat_keluarga === 'Ya') {
        $prob['Hipertensi'] *= 1.4;
    } else {
        $prob['Normal'] *= 1.2;
    }

    // Faktor 4ï¸âƒ£: Keluhan
    if (strpos($keluhan, 'pusing') !== false || strpos($keluhan, 'jantung') !== false) {
        $prob['Hipertensi'] *= 1.3;
    }
    if (strpos($keluhan, 'lemas') !== false) {
        $prob['Rendah'] *= 1.2;
    }
    if (strpos($keluhan, 'sesak') !== false) {
        $prob['Pre-Hipertensi'] *= 1.1;
    }

    // Pilih hasil dengan peluang terbesar
    arsort($prob);
    $hasil = key($prob);

    // Tentukan pesan hasil diagnosa
    switch ($hasil) {
        case 'Rendah':
            $pesan = "Tekanan darah rendah. Perbanyak istirahat dan konsumsi air putih.";
            break;
        case 'Normal':
            $pesan = "Tekanan darah normal. Pertahankan gaya hidup sehat!";
            break;
        case 'Pre-Hipertensi':
            $pesan = "Tekanan darah mulai meningkat. Jaga pola makan dan rutin berolahraga.";
            break;
        case 'Hipertensi':
            $pesan = "Tekanan darah tinggi. Segera konsultasikan ke dokter.";
            break;
    }

    return [$hasil, $pesan];
}

| **Faktor Riwayat Keluarga**                                      | Jika ada riwayat keluarga hipertensi, maka peluang kategori *Hipertensi* meningkat.                                                                                                |

| **Bagian `switch`**                                              | Menentukan pesan yang akan ditampilkan sesuai hasil akhir, seperti saran medis atau gaya hidup.                                                                                    |
| `return [$hasil, $pesan];`                                       | Mengembalikan hasil diagnosa dan pesan ke kode utama agar bisa ditampilkan di modal.                                                                                               |
